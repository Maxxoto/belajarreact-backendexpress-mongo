# BelajarReact Todo App 

First project using ReactJS Todo App (https://belajar-react.herokuapp.com/)

## Project Overview


- MongoDB Hosted in [MongoDB Atlas Cloud](https://www.mongodb.com/cloud/atlas)
- Client & Server API Hosted in [Heroku](https://www.heroku.com)
- [ReactJs](https://github.com/facebook/react)
- [Bootstrap](https://getbootstrap.com/)
- [ExpressJs](https://expressjs.com/)

## Usage Testing API


- Download [Postman](https://evening-badlands-83186.herokuapp.com/)
- Copy https://evening-badlands-83186.herokuapp.com/todos (Using GET)
- Copy https://evening-badlands-83186.herokuapp.com/create/add (Using POST and Fill JSON to Insert)
- Copy https://evening-badlands-83186.herokuapp.com/edit/{id} (Using POST and change {id} with id what do you want to change)
